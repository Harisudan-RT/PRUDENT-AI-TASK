# PRUDENT AI Internship Task Submission

## Overview
This repository contains the completed tasks assigned as part of the Prudent AI internship technical assessment.  
Each task focuses on a different aspect of data processing, automation, and intelligent system design.

---

## Task 1 — Transaction Value Extractor
### Objective
Develop a Python-based component to extract structured transaction data from unstructured text logs.

### Key Features
- Extracts transaction type, amount, and ID using pattern recognition and regex.
- Returns clean and validated transaction tuples in JSON format.
- Handles mixed and inconsistent text formats gracefully.
- Includes error handling and modularized logic for scalability.

---

## Task 2 — Bank Statement Parser (Gemini)
### Objective
Build a bank statement parser that integrates with Google Gemini to analyze PDF or image statements.

### Key Features
- Extracts all key transaction and summary fields into a structured JSON output.
- Utilizes Gemini’s generative AI capabilities for field recognition and interpretation.
- Generates concise financial insights and spending summaries.
- Supports both PDF and image inputs with OCR-based fallback.
- Outputs include structured data and AI-driven insights.

---

## Output
- Each task generates structured, human-readable JSON outputs.
- Insights and summaries provide clarity on financial health indicators.
- Console-friendly and API-compatible design for easy integration.

---

## Tech Stack
- **Language:** Python  
- **Libraries:** pdfplumber, pytesseract, google-genai, re, json, Pillow  
- **AI Model:** Google Gemini (Generative AI)  
- **Tools:** Jupyter Notebook, Visual Studio Code
